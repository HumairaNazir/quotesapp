# Quotes App

A simple and clean Flutter application that displays inspirational
quotes.

## Features
- Beautiful and minimal UI
- Inspirational quotes collection
- Offline support
- Easy to use interface

## Tech Stack
- Flutter
- Dart


## Project Purpose
This project is created to practice Flutter development 
and demonstrate UI building and basic app structure.

## How to Run the Project
1. Clone the repository
2. Run `flutter pub get`
3. Run `flutter run`
